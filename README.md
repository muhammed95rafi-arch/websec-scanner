# WebSecAudit — Security Assessment Tool

A web-based security scanner built for the Qnayds Academy webinar.

## Features
- Auto-detects 13 security checks (headers, HTTPS, cookies, CORS)
- 62 manual verification items with override support
- PDF report export
- Dark-themed professional UI

## Run Locally
```bash
pip install -r requirements.txt
python app.py
```
Then open: http://localhost:5000

## Stack
- Python / Flask
- HTML + CSS + JavaScript
- jsPDF (PDF export)
